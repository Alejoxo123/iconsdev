Iconos para Desarrolladores

Recursos libres de iconos para tus aplicaciones y proyectos web

Características del Proyecto

Acceso Sencillo: Encuentra iconos por nombre y categoría con nuestra intuitiva API.

Filtrado y Búsqueda: Busca iconos por nombre, categoría y más.

Tecnologías Usadas: Construido con Angular 17, TailwindCSS y la API de Iconfinder.
